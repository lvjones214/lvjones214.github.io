Virtual Pet

Created Virtual Pet Simulator console application using java. The simulator creates both robotic and organic pets, allows the user to feed the pets, water the pets and play with the pets. The user can also put pets into the a Virtual Pet Shelter and take them out again. This was done as a group project with two other students in the We Can Code It boot camp.
